# MyCn18: IDE para Desarrollo y Ejecución de Código con IA

**MyCn18** es un entorno de desarrollo (IDE) diseñado para construir, auditar y ejecutar lógica de programación apoyada por inteligencia artificial. A diferencia de otros editores, MyCn18 permite visualizar el proceso de pensamiento de la IA en tiempo real, usar un tablero Kanban para la gestión de tareas, simular resultados de decisiones a medida que la IA trabaja y visualizar diff de cambios mediante un diagrama de código base.

### Estado actual

MyCn18 es un proyecto personal que está en fase inicial de desarrollo. El objetivo es crear una herramienta que yo mismo, como desarrollador que prefiere el código a las interfaces limitadas, quiera usar todos los días.
